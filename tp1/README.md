Para las pruebas pasar el archivo a ejecutable con el comando
´´´ chmod -x pruebas.sh´´´

Tienen que existir los archivos con nombres
´´´ 
test_unix2dos_1.txt 
test_dos2unix_1.txt

out_unix2dos_1.txt
out_dos2unix_1.txt
´´´ 
los numeros van el 1 al 5, si se hacen menos pruebas modificar el script

Finalmente se corre con 
´´´
./pruebas.sh unix2dos
o
./pruebas.sh dos2unix
´´´
